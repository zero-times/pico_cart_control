Pico GP26 电池电压检测分压电路文档
1. 目的

这个电路用于让 Pico 读取小车 24V 电池电压。

Pico 的 GP26 是 ADC 输入，只能读取 0~3.3V。
所以不能直接接 24V 电池，必须先通过电阻分压，把 24V 压到 3.3V 以下。

目标：

24V 电池
  ↓
电阻分压
  ↓
2.xV 模拟电压
  ↓
Pico GP26 / ADC0
  ↓
Pico 计算真实电池电压
  ↓
Android App 显示电压 / 电量
2. 推荐电路：保护版

推荐元件：

100k 电阻 ×1
10k 电阻 ×1
1k 电阻 ×1
0.1uF 电容 ×1
100mA~250mA 小保险丝 ×1，推荐

完整电路：

总开关后 24V+
     │
  100mA~250mA 小保险丝
     │
    100k
     │
     ├──── 1k ─── ADC_OUT ─── Pico GP26
     │              │
    10k            0.1uF
     │              │
系统 GND ──────────┴────── Pico GND

简化理解：

24V+ ── 100k ── 分压点 ── 10k ── GND
                  │
                 GP26 只读取这里的电压

重点：
GP26 不是串在主电路里，它只是接在分压点上读电压。

3. 不要这样接

错误接法：

24V+ ───── GP26

这个会直接烧 Pico。

也不要理解成：

24V+ → 100k → GP26 → 10k → GND

正确理解是：

24V+ → 100k → 分压点 → 10k → GND
                 │
                GP26

GP26 只是一个高阻输入，相当于万用表表笔，读取分压点电压。

4. 分压比例

使用：

上方电阻：100k
下方电阻：10k

分压公式：

ADC_OUT = 电池电压 × 10k / (100k + 10k)
ADC_OUT = 电池电压 / 11

常见电压对应：

电池电压	GP26 电压
22.0V	2.00V
24.0V	2.18V
26.0V	2.36V
29.4V	2.67V
36.0V	3.27V

所以这个比例适合 24V 锂电系统。

5. 推荐接线位置

建议从 总开关后 取样：

电池+
  ↓
40A 主保险丝
  ↓
总开关
  ↓
小保险丝
  ↓
100k 分压电阻

这样总开关关闭后，分压电路也断电，不会长期耗电。

不建议第一版直接接电池正极常供电。

6. 和外置电量表并存

外置电量表可以继续接：

总开关后 24V+ ─── 电量表+
系统 GND        ─── 电量表-

Pico 分压电路也接同一组电源：

总开关后 24V+ ─── 分压板 BAT+
系统 GND        ─── 分压板 GND
分压板 ADC_OUT  ─── Pico GP26

两者互不影响。

外置电量表负责肉眼查看。
Pico GP26 负责后续 App 显示和低电保护。

7. 建议做成小分压板

不要裸飞线焊到 Pico 上。

建议做一个小板，板上有 3 个端子：

BAT+      接总开关后的 24V+
GND       接系统 GND
ADC_OUT   接 Pico GP26

小板内部：

BAT+ ── 小保险丝 ── 100k ── 分压点 ── 10k ── GND
                              │
                             1k
                              │
                           ADC_OUT
                              │
                            0.1uF
                              │
                             GND

做好后用热缩管或者小盒子包起来，固定在设备舱里。

8. 元件确认清单

你后面拆元件时，可以确认这些：

元件	目标值	备注
上方电阻	100k	接 24V+ 那边
下方电阻	10k	接 GND 那边
GP26 串联电阻	1k~4.7k	保护 ADC
电容	0.1uF	ADC_OUT 到 GND
小保险丝	100mA~250mA	采样支路保护

电阻建议用 1% 精度。
普通 1/4W 电阻就够。

9. 上电前检查
9.1 不接电池、不接 Pico，先测电阻

用万用表电阻档测：

测量位置	正常值
BAT+ 到 GND	约 110k
BAT+ 到 ADC_OUT	约 101k
ADC_OUT 到 GND	约 11k

如果测到：

BAT+ 到 ADC_OUT 接近 0Ω

说明接错，不能接电池。

9.2 接电池，但先不要接 Pico

先只接电池和分压板，不接 Pico。

用万用表测：

ADC_OUT 对 GND 的电压

正常应该小于 3.3V。

参考：

电池电压	ADC_OUT
24V	约 2.18V
29.4V	约 2.67V
36V	约 3.27V

只有确认 ADC_OUT < 3.3V 后，才能接 Pico GP26。

10. Pico 读取代码
from machine import ADC
import time

BAT_ADC = ADC(26)  # GP26 / ADC0

R_TOP = 100_000.0
R_BOTTOM = 10_000.0
ADC_REF = 3.3
ADC_MAX = 65535

DIVIDER_RATIO = (R_TOP + R_BOTTOM) / R_BOTTOM  # 11.0

def read_battery_voltage(samples=20):
    total = 0
    for _ in range(samples):
        total += BAT_ADC.read_u16()
        time.sleep_ms(2)

    raw = total / samples
    adc_v = raw / ADC_MAX * ADC_REF
    bat_v = adc_v * DIVIDER_RATIO
    return bat_v

后续可以加移动平均，避免电机启动时电压波动太明显。

11. 风险点

重点避免：

1. 24V 正极直接碰到 GP26。
2. 分压板裸露，震动后短路。
3. ADC_OUT 未测电压就直接接 Pico。
4. BAT+ 和 GND 接反。
5. ADC 采样线和电机大电流线长距离绑在一起。

建议：

分压板做好绝缘。
ADC_OUT 线尽量短。
采样线远离电机线。
上电前一定先用万用表测 ADC_OUT。
先不接 Pico 测试，确认没问题再接 Pico。
12. 当前建议

你的 V1 可以这样走：

第一步：
只装外置电量表，人工看电量。

第二步：
拆元件，确认 100k / 10k / 1k / 0.1uF 是否齐。

第三步：
焊一个独立分压小板。

第四步：
不接 Pico，先用万用表测 ADC_OUT。

第五步：
确认 ADC_OUT 小于 3.3V 后，再接 Pico GP26。

后续如果想要电流、功率、Wh 统计，再上：

INA226 + 50A/75mV 外置分流器
13. 最终结论

这个分压电路的核心是：

100k 和 10k 先把 24V 压成 2.xV
GP26 只读取这个 2.xV

推荐最终电路：

总开关后 24V+
   │
小保险丝
   │
100k
   │
├── 1k ── GP26
│         │
│        0.1uF
│         │
10k      GND
│
GND

焊好之后，先测 ADC_OUT 对 GND，小于 3.3V 再接 Pico。在代码里加个小小的 10ms 延时，之后读取到的电压就是稳如泰山的 2.67V 真实分压值了
